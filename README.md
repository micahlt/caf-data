# Caf Data
This dashboard is a [Next.js](https://nextjs.org/) webapp that lets you visualize long-term historical data from [The Caf App](https://thecaf.app).  View the site at [data.thecaf.app ‚ü∂](https://data.thecaf.app)

## Development

Install all dependencies and run the development server:

```bash
npm i
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
